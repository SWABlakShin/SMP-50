# What this project is:
This is a file for Super Mario Paint, a fan-made project
http://download.supermariopaint.com (it's a github link).
I'm putting it here because it's a text file, and github likes those a lot, plus if anyone wants to listen to it, that's cool, too.
I don't feel like recording it, so this is the closest to 'putting it out there' that I'm willing to go through.

# To break it down...
This is the song *50%* by Official Hige Dandism, a JPOP band, in a project similar to Mario Paint.
Originally, I was going to use Nintendo Switch Online's Mario Paint on the SNES to 'natively' (on an emulation) create the song in the game;
however, there isn't enough tools nor power to accurately recreate it, so I found this program.

# The exact file included is...
a .txt file that includes the number of notes of each type I used, the beats (which is high for a reason)
and the sharps and flats of the notes (which was not present in the original version).
